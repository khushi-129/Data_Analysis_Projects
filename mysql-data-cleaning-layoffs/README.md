# MySQL Data Cleaning - Layoffs Dataset

## Overview
Performed end-to-end data cleaning using MySQL on a real-world layoffs dataset.

## Skills Used
- CTEs
- Window Functions
- ROW_NUMBER()
- Data Standardization
- NULL Handling
- Duplicate Removal

## Steps Performed
1. Removed duplicates
2. Standardized data
3. Handled missing values
4. Converted date datatype
5. Removed unnecessary rows and columns
